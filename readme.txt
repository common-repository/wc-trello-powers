Woo & Trello Powers

== Description ==

Com este plugin você pode criar cartões no Trello através dos gatilhos Woocommerce

== HOW IT WORKS ==

Woo & Trello Powers usa Trello API com sua chave Trello API. Quando há um Novo Pedido ou uma Mudança de Status do Pedido do Woocommerce, este plugin pega as informações do pedido para criar um card.

== Standard Features ==

- Autentificação com um click;
- Crie cartões para diferentes ações de Woocommerce;
- Selecione exatamente qual lista você deseja integrar;

== Installation ==

1. Upload `woo-trello-powers` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Get Trello access Key and Past it . 
4. Select a trigger and Trello list Where you want to create Cards.
5. Change the Settings by your Preference .enjoy
